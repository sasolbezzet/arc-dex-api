# Privacy Roadmap

ARCOX Pay today:

- Public USDC invoice/payment link on Arc Testnet.
- Public transaction hash.
- Sender and recipient visibility depends on the chain.

Future ARCOX Private Pay ideas:

- Private invoice.
- Private payment history.
- Private merchant balance.
- Private B2B payments.
- Payroll-like private flows.
- Shield USDC into Arc Privacy Sector / ArcaneVM when available.
- Pay inside a private execution environment.
- Authorized query for buyer and merchant.
- Unshield when merchant wants public funds.

Important:

- Private payments are not live.
- ARCOX does not fake private transactions.
- Arc privacy features are roadmap-only until available to builders.
- Current implementation remains public.
